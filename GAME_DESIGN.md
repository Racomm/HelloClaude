# 醉翁觅酒 · 游戏设计文档

## 游戏概述
关卡制轻度休闲游戏。玩家扮演一个爱酒成痴的醉翁，通过收集从天而降的金币和道具，凑够钱购买每一关对应的酒。

## 技术栈
- 原生 JavaScript (ES6+)
- HTML5 Canvas
- CSS3
- PWA（后续可接入 manifest）

---

## 核心玩法
- 角色可**左右移动**和**跳跃**
- 物品从屏幕顶部随机位置落下
- 触碰正向道具 → 增加财富值
- 触碰炸弹 → 损失一条命（有短暂无敌帧）
- 财富值达到目标价格 → 通关，进入下一关

## 角色与操作

- **键盘**：← / A 左移，→ / D 右移，空格 / ↑ / W 跳跃
- **触屏**：左三分之一按住左移，右三分之一按住右移，中间点击跳跃
- 角色左右移动到屏幕边缘时**穿越至对侧**
- 角色有**醉酒摇晃动画**（旋转摆动）

---

## 生命系统
- 初始 3 条命（❤️❤️❤️）
- 碰到炸弹 -1 命，触发无敌帧（约 1.5 秒）
- 命数归零 → 游戏结束
