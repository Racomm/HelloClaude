# 🦖 Chrome Dino Game - iPhone 15 Plus Edition

一个彩色版的 Chrome 小恐龙跑酷游戏，专为 iPhone 15 Plus 优化。

## ✨ 特性

- 🎨 **彩色图形** - 丰富的颜色和动画效果
- 📱 **iPhone 15 Plus 优化** - 完美适配 6.7 英寸屏幕
- 🌐 **PWA 支持** - 可安装到主屏幕，离线游戏
- 🎮 **触控优化** - 专为触摸屏设计的操作方式
- 🌓 **昼夜模式** - 自动切换昼夜场景
- 🔊 **音效系统** - 完整的游戏音效

## 🎮 游戏玩法

### 桌面端
- **空格键 / 上箭头** - 跳跃
- **下箭头** - 下蹲
- **Shift + D** - 调试模式

### 移动端
- **点击上方** - 跳跃
- **点击下方** - 下蹲（按住保持下蹲）
- **横屏游戏** - 获得最佳体验

## 📱 在 iPhone 上安装

1. 用 Safari 访问游戏网址
2. 点击分享按钮 📤
3. 选择"添加到主屏幕"
4. 完成！现在可以离线玩了

详细安装说明请查看 [PWA-INSTALL.md](PWA-INSTALL.md)

## 🛠️ 技术栈

- 原生 JavaScript (ES6+)
- HTML5 Canvas
- CSS3 动画
- Service Worker (PWA)
- Web Audio API

## 📦 本地运行

```bash
# 克隆仓库
git clone https://github.com/Racomm/HelloClaude.git
cd HelloClaude

# 启动本地服务器
python3 -m http.server 8000

# 访问
open http://localhost:8000
```
